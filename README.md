# AI-Powered Smart Glasses for Blind Users

## Overview

This project aims to help visually impaired individuals by providing:

- Object Recognition
- Text Reading (OCR)
- Voice Assistance

The system captures images using a camera and processes them using Artificial Intelligence.

## Features

- Real-time object detection
- OCR-based text reading
- Audio feedback
- Low-cost wearable design

## Hardware Used

- Arduino UNO
- ESP32-CAM
- DFPlayer Mini
- Li-ion Battery
- Speaker/Earphones

## Software Used

- Python
- OpenCV
- YOLOv8
- EasyOCR
- Arduino IDE

## Workflow

Camera → AI Processing → Object/Text Recognition → Speech Output

## Future Scope

- Face Recognition
- GPS Navigation
- Currency Detection

- Mobile App Integration
